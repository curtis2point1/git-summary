# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.2.x   | :white_check_mark: |
| < 1.2   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do NOT** open a public issue.
2. Report it through GitHub private vulnerability reporting: https://github.com/curtis2point1/git-summary/security/advisories/new
3. Include a detailed description of the vulnerability and steps to reproduce.

You can expect:
- Acknowledgment within 48 hours
- Regular updates on the fix progress
- Credit in the release notes (unless you prefer to remain anonymous)

Thank you for helping keep git-summary secure!
